# Lite Aeronautics

Minecraft 1.21.1 (NeoForge) modpack — chủ đề Create Aeronautics + Cataclysm.

## Cảnh báo bản quyền
Tất cả file `.jar` trong thư mục `mods/` thuộc về tác giả gốc của từng mod.
Modpack này được chia sẻ cho mục đích cá nhân / học tập. Vui lòng tải mod từ
nguồn chính thức (CurseForge / Modrinth) nếu bạn là tác giả mod và không đồng ý
với việc redistributet. Toàn bộ credit thuộc về các tác giả mod tương ứng.

## Cấu trúc
- `mods/` — mod (NeoForge 1.21.1)
- `config/`, `defaultconfigs/`, `kubejs/` — cấu hình pack
- `manifest.json` — CurseForge manifest
- `saves/`, `logs/`, `crash-reports/`, `showdown/` bị loại khỏi repo (xem `.gitignore`)

## Cài đặt
Dùng launcher hỗ trợ CurseForge (Prism/Vortex/CurseForge) hoặc copy instance vào
`.minecraft` rồi chạy NeoForge 1.21.1.
